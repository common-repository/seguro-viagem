## Description

O Plugin da [Real Seguro Viagem](https://www.seguroviagem.srv.br/) permite que você adicione nossa ferramenta comparativa de seguro viagem - um hotsite - ao seu site WordPress e aumente seus lucros. 🚀

Com o nosso novo plugin, você pode adicionar um comparador de seguro viagem personalizado ao seu blog WordPress. O hotsite é totalmente responsivo e se adapta a qualquer tamanho de tela, garantindo uma experiência perfeita para seus leitores.

O hotsite é feito de forma fácil e intuitiva direto na sua Plataforma de Indicadores, o que significa que você não precisa se preocupar com programação ou design. Basta adicionar suas informações e pronto!

## Installation

### Como usar o plugin

1. Faça o seu cadastro no [Programa de Indicadores da Real](https://app.seguroviagem.srv.br/indica) 🚀
2. Instale o plugin no seu blog WordPress. ⬇️
3. Acesse o Indica, o painel dos Indicadores da Real e vá para **Divulgue > Integrar > Hotsite.** 🏃🏼‍♂️
4. Personalize o seu comparador. ✨
5. Acesse o WP-Admin em “Seguro Viagem” informe seu código de afiliado. ✅
